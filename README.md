## Run Locally

Clone the project

```bash
  git clone https://github.com/rgilsimoes/csw-office-finder
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Install Ionic cli

```bash
  npm install -g @ionic/cli
```

Start the server

```bash
  ionic serve
```
